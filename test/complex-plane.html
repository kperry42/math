The test data has been evaluated in unit increments from -49.5 to 49.5 in both directions on the the complex plane. A green dot above indicates that the same calculation with this library matches the test data to ten significant digits. A yellow dot indicates a variance of two digits in the significand, i.e. the part of the value in scientific notation without the exponent. A red dot indicates a variation larger than two digits, where again this difference in the significand. A black dot is either a serious error or NaN.

Functions requiring more that one argument, such as Bessel or elliptic functions, indicate the additional parameters in the function name in square brackets. Most of these parameters precede the main argument in the function call. When they do not an empty list is included in the file name to allow detection of these cases.

These tests give an immediate visual sense of the accuracy of this library. They can also be used to identify argument domains that need attention.
